# 🧾 小票情绪生成器

把你的关系变成一张可以分享的票据。

## 功能
- ✏️ 自定义店铺名和商品/价格
- 🔗 生成分享链接（URL 编码数据）
- 📥 导出小票为 PNG 图片

## 本地开发

```bash
npm install
npm run dev
```

## 部署到 Vercel

1. 把项目上传到 GitHub
2. 打开 [vercel.com](https://vercel.com) → Import Project
3. 选择 GitHub 仓库
4. 框架选 **Next.js**（会自动识别）
5. 点击 Deploy

完成！Vercel 会自动给你分配域名。
